# Interactive Cyber Security & IT Audit Portfolio

Welcome to my personal interactive resume and professional dashboard. This project acts as a live, secure gateway to my career progression, technical competencies, and academic background.

👉 **Live Site:** [https://andreas156.github.io/](https://andreas156.github.io/)

## 🛠️ Core Focus & Architecture
This portfolio is built as a responsive, single-page web application featuring an interactive "Secure Access Gateway" splash screen that transitions into a clean, modern executive dashboard. 

* **Frontend:** Semantic HTML5, Modern CSS3 (Grid/Flexbox Layouts, Tailwind-inspired color token system, custom keyframe transitions), and Vanilla JavaScript for runtime UI interactions.
* **Design Philosophy:** Optimizing heavy text payloads (such as extensive auditing records, publications, and technical certifications) into a highly legible, scannable, dark-mode dashboard ecosystem.
